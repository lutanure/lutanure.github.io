dotTrack

A SwiftUI wellness tracking application with interactive visualization dashboards and historical trend analysis to help users understand their habits over time.

View on GitHub →

Overview

dotTrack is an iOS wellness tracking app built with SwiftUI. It lets users log daily habits and wellness metrics, then surfaces that data through clean visualization dashboards — bar charts, streaks, and historical trend lines — so patterns become easy to spot at a glance.

The app was designed with a minimal, dot-grid-inspired aesthetic (hence the name) and focuses on making data entry as frictionless as possible, so users actually stick with logging.

  • Custom SwiftUI views for habit logging and daily check-ins
  • Visualization dashboards using Swift Charts for trend analysis
  • Persistent local storage via Core Data
  • Streak tracking and summary statistics per metric

Tech Stack

Built natively for iOS using Apple's modern declarative UI framework.

## Features

The site includes:

- **Interactive timeline** of my academic and professional journey
- **Projects section** showcasing software work
- **Responsive design** optimized for desktop and mobile
- **Bilingual content (English / Portuguese)** using an i18n system

## Tech Stack

This portfolio is intentionally lightweight and built with:

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages

No frameworks or build tools are used to keep the site simple and fast.
